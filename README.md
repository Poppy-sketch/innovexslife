## Usage Instructions

### Local Testing
1. Extract all files to a folder
2. Open `index.html` in any modern web browser
3. Navigate through pages using the menu
4. All features work offline (except external images/videos)

### Web Hosting
1. Upload all files to your web server
2. Ensure file structure is maintained
3. No server-side processing required
4. Works with any static hosting (GitHub Pages, Netlify, Vercel, etc.)

### Customization
- **Colors**: Edit CSS variables in `styles.css` (`:root` section)
- **Content**: Modify HTML content directly in page files
- **Images**: Replace Unsplash URLs with your own images
- **Videos**: Update YouTube embed IDs in video sections
- **Logo**: Change logo text/icon in header sections
